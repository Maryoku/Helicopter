Необходимо разработать программу для расчета данных по траектории полета вертолета.

Вертолет в определенной последовательности облетает населенные пункты. В каждый из которых он должен прибыть точно к определенному времени.

В каждом пункте он тратит 5 минут на посадку, выгрузку и взлет.

В каждой строке входного файла находится информация об очередном населенном  пункте в маршруте вертолета:
X    Y    HH:MM
Где (X;Y) - координаты населенного пункта - вещественные числа
HH:MM - часы и минуты, когда вертолет должен прилететь в точку (время считается относительно времени вылета 00:00 из нулевой отметки с координатами (0;0))

Пример входного файла:
50,34    100,5    00:45
40,0     220,78    00:55
120,6    100,9    01:30
170,3    350,54    02:45

По входным данным о маршруте вертолета необходимо:
1. Вычислить длину каждого отрезка маршрута и среднюю скорость полета на каждом отрезке пути
2. Определить имеются ли в траектории полета между населенными пунктами циклы - отрезки пути, когда вертолет вновь посещает какой-либо населенный пункт. Для каждого из таких найденных циклов в траектории необходимо вывести координаты пунктов, которые его образуют.
3. Определить имеются ли в траектории полета пересекающиеся отрезки маршрута. Для каждой точки пересечения отрезков вывести координаты точки пересечения и номера пересекающихся отрезков.

Требования к разработке:
1. Необходимо использовать структуры данных (struct)
2. Грамотная декомпозиция: программа должна быть разделена на несколько модулей (модули исходного кода и заголовочные файлы), в каждом модуле описаны функции и необходимые структуры.
3. Можно использовать возможности STL
4. Проверка входных данных при считывании на корректность
5. Удобный интерфейс пользователя
6. Следование рекомендациям по стилю написания программ на С++ (именование, форматирование, оформление кода)
